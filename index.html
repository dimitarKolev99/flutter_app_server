Intro

I am a student of International Media and Computing at HTW Berlin, currently at the end of my studies.

After acquiring fundamental knowledge in object-oriented programming with Java and Javascript as well as an introduction in SQL databases, I learned useful design patterns and clean code practices. Then I chose to specialize in Web Technology so I took a course on Web Development where I had to build a web application utilizing the MVC pattern. I learned how to make a NodeJS server, which would serve my HTML/CSS views on the client-side, and store data in a nosql database.

This semester I worked at idealo internet GmbH as a Project Student. We had to build a cross-platform mobile application using Google's Flutter framework and the Dart programming language. Originally, the project was meant to be a mobile application but after the release of the mobile version, I decided to work further on it and turn it into a web application aswell. Below I go on to explain the development of the web version in more detail.

Penny Pincher

The idea of the application is to present the user discounted products.

The main feature is the Live Feed where the user receives new bargains in real time.

Furthermore, there is a filtering option and browser page where one can choose products of specific category, price range and discount percent.

Tech Stack

To build the the frontend, we chose to use Flutter, as it delivers both Android and iOS applications with a single code base. I decided to also use it for the web version because I could reuse a big part of the code.

The backend of the application is a Node and Express server that exposes an interface which is used by the frontend.

To store the data, I chose MongoDB because it uses the convenient JSON-format.

We used the Git version control system to collaborate in the team. Initially I deployed the app on Heroku which led to some problems so I decided to use GitHub Pages at the end.
